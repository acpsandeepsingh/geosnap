# **App Name**: GeoSnap

## Core Features:

- Photo Capture: Capture photos using the device's camera.
- GPS Data Overlay: Overlay GPS coordinates and timestamp onto live camera which is shown after capturing
- Location Display: Display the current location on a map.
- Photo Organization: Allow users to organize captured photos in folders based on date/location.

## Style Guidelines:

- Primary color: Dark slate blue (#374785) for a professional and reliable feel.
- Background color: Very light gray (#F0F4F8).
- Accent color: Sky blue (#5885AF) for interactive elements and highlights.
- Body and headline font: 'Inter' sans-serif for a modern, objective look.
- Use minimalist, outline-style icons to represent map markers, folders, and camera functions.
- Simple, intuitive layout with clear information hierarchy, with camera view occupying most of the screen.
- Smooth transitions for screen changes and displaying GPS data.
- Modern design aesthetic.
- Menu should have a modern look.
- Default page opens as a camera app.
- run on normal html server
